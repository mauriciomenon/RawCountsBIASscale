# SCADA Analog Scaling Calculator

A desktop calculator for adapting analog measurement values used in SCADA systems.
The tool calculates BIAS, SCALE, raw counts, and hexadecimal values for analog
data conversion between field instrumentation ranges and SCADA/RTU engineering
values.

The project is intended for workflows involving SCADA analog point scaling,
including IEC 60870 based environments. It can support commissioning,
validation, and troubleshooting scenarios in systems such as ABB/Hitachi SCADA
NMR3 and NMR5, with RTUs connected through Harris/GE infrastructure, for
example.

This repository contains the active working version of the calculator, plus
archived historical code kept separately for traceability.
